🎮 Rock, Paper, Scissors Game

This is a simple and interactive Rock, Paper, Scissors game built using HTML, CSS, and JavaScript.
The game runs directly in the browser and allows a user to play against the computer with smooth animations and real-time score updates.

The main goal of this project was to practice DOM manipulation, animations, and clean UI design without using any external libraries or frameworks.

✨ What Makes This Version Different?

Full screen animated GIF background

Continuous smooth rock shaking animation before every round

Animation stops instantly when the user makes a choice

Computer rock appears mirrored

Paper emoji remains vertical (not rotated)

Automatic reset after each round

Real-time score counter

Reset button to clear the game

Clean separation of HTML, CSS, and JavaScript

🛠 Technologies Used

HTML5

CSS3 (Flexbox, Keyframes, Animation)

Vanilla JavaScript (DOM Manipulation & Game Logic)

No frameworks or libraries were used.

🚀 How to Run the Game

Download or clone this project.

Place all files inside the same folder.

Add your GIF background file.

Open index.html in your browser.

That’s it. No installation required.

🎮 How the Game Works

When the page loads, both user and computer start with a shaking rock animation.

The user selects Rock, Paper, or Scissors by clicking one of the emoji buttons.

The computer randomly chooses one option.

The animation stops and both results are shown.

The winner is displayed on the screen.

The score updates automatically.

After a short delay, the game resets and starts shaking again.

🧠 Game Rules

Rock beats Scissors

Paper beats Rock

Scissors beats Paper

Same choice = Draw

The computer choice is generated randomly using JavaScript.
